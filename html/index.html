Hex
scattering database
“Hex” stands for “Hydrogen-electron collisions”. This acronym actually covers several distinct things:
  • low-energy scattering code implementing direct solution of Schrödinger equation using exterior complex scaling workaround for boundary condition, a.k.a. hex-main,
  • high-energy scattering code implementing first and second order of distorted wave Born approximation, a.k.a. hex-dwba,
  • the database of T-matrices, as computed by these two modules, and
  • the user interface (both CLI and Web), a.k.a. hex-db, which uses these data to compute derived scattering quantities.
The package is maintained by the Institute of Theoretical Physics at the Charles University in Prague, Czech Republic, in the collaboration with the Astronomical Institute of the Czech Academy of Sciences.
